# Category Images

This directory should contain 4 category images for note categorization.

## Required Images:
- `category_0.png` - Category image 1
- `category_1.png` - Category image 2
- `category_2.png` - Category image 3
- `category_3.png` - Category image 4

## Image Specifications:
- Format: PNG (recommended) or JPG
- Recommended size: 100x100 pixels or similar square dimensions
- These images will be displayed in note cards and during note creation/editing

## Usage:
The images are referenced by index (0-3) in the NoteModel's `categoryImageIndex` field.
